# Wen-Chieh Lee的生活

This repository contains the source code for the "Wen-Chieh Lee的生活" website, built with [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).

The website is a personal blog and record of Wen-Chieh Lee's life, focusing on the following topics:

*   **生活 (Life):** Thoughts and shares from daily life.
*   **種植 (Planting):** Experiences and notes on planting.
*   **養殖 (Breeding):** Records and stories about animal breeding.
*   **釀酒 (Winemaking):** Adventures in homebrewing.
