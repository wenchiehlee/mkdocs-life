# 關於本站

"Wen-Chieh Lee的生活" 是一個記錄和分享生活點滴的個人網站，主要圍繞以下幾個主題：

*   **生活 (Life):** 日常生活的感悟與分享。
*   **種植 (Planting):** 關於植物種植的經驗與心得。
*   **養殖 (Breeding):** 養殖動物的記錄與趣事。
*   **釀酒 (Winemaking):** 家庭釀酒的嘗試與分享。
